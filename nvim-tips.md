# Vim Scrolling Mneumonic

Shamelessly stolen from <https://stackoverflow.com/a/60607822>

```
+-------------------------------+
^                               |
|c-e (keep cursor)              |
|H(igh)             zt (top)    |
|                   ^           |
|           ze      |      zs   |
|M(iddle)  zh/zH <--zz--> zl/zL |
|                   |           |
|                   v           |
|L(ow)              zb (bottom) |
|c-y (keep cursor)              |
v                               |
+-------------------------------+
```

Additionally, `C-d` and `C-u` are page-Down and page-Up.
